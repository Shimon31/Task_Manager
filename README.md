# Task Manager

Task Manager is an Android application built with **Kotlin** and **MVVM architecture**. The app helps users manage tasks efficiently with a simple and clean user interface.

---

## Core Features

- Create, read, update, and delete tasks.
- Task properties:
  - Title (required)
  - Description (optional)
  - Priority: Low, Medium, High
  - Status: To Do, In Progress, Done
  - Due date (optional)
  - Created timestamp

---

## User Experience

- View all tasks with filtering options.
- Search tasks by title.
- Sort tasks by date, priority, or status.
- Offline support: All data stored locally.
- Simple and clean UI for easy navigation.
